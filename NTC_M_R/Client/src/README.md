# This folder contain the FRONTEND REACT project files.

## Requirement
    version lts :  node >=16
## Run the following commands :
  ### Run using Express -
  ##### Navigate to the below path :-
  ```
      consumer/src/client/
   Run the command :-
      npm run build
   Navigate to the below path :-
      consumer/src
   Run the command :
      npm start
```
##### Run without Express -
   Navigate to the below path :-
   ```
      consumer/src/client/
   Run the command :
         npm start
   ```
